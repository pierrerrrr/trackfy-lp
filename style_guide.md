# Trackfy Style Guide

## Color Palette

| Color Name | Hex Code | Usage |
| :--- | :--- | :--- |
| **Primary Green** | `#42E36B` | Main actions, highlights, branding |
| **Secondary Dark Green** | `#194D2F` | Secondary elements, subtle backgrounds |
| **Accent White** | `#FFFFFF` | Text, icons on dark backgrounds |
| **Background Black** | `#141414` | Main page background |

## Typography

**Font Family**: Poppins

- **Heading 1**: Bold / ExtraBold
- **Heading 2**: SemiBold
- **Heading 3**: Medium
- **Body Text**: Regular

## Logo Usage

- **Clear Space**: Maintain clear space around the logo.
- **Minimum Size**: Ensure legibility.
- **Incorrect Applications**: Do not distort or place on clashing backgrounds.
